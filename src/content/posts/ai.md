# Perplexity
::link{url="https://www.perplexity.ai/" text="Perplexity" icon="https://favicon.im/perplexity.ai"}
Perplexity 是由 **Perplexity AI** 開發的「**知識與推理導向 AI 助理（reasoning & knowledge assistant）**」，不只是回答問題的語言模型，而是一個能夠理解、蒐集、分析資訊並生成完整解答的 AI 智能系統。  
## Perplexity 是什麼
- 定位：專注於**知識檢索與推理整合**的雲端 AI 助手，能閱讀多來源內容、整合背景脈絡，產出有根據、具條理的回應。  
- 模型：底層採用 **GPT‑5 系列的推理強化模型**，結合檢索（retrieval）、多階段推理（multi‑step reasoning）與自然語言生成（NLG）能力，支援深度知識工作。  
- 存取方式：可透過 Perplexity 網站、行動 App，或透過整合 API、插件與 Copilot 模式使用。
## 能做哪些事
- 資訊搜尋與整合：跨多個資料來源分析、摘要、比較，給出具脈絡的完整回答。  
- 理解與解釋：針對技術概念、學術主題、程式碼運作邏輯等進行逐步說明或可視化分析。  
- 專案協作：理解使用者的上下文與需求，協助規劃、撰寫或檢討內容（如文件、報告、專案規格）。  
- 學習輔助與決策支持：整理重點、生成學習路徑，幫助使用者做出選擇或制訂行動計畫。  
- 跨技能應用：具備多模態與外部工具整合能力，可分析圖像、生成資料表、或結合外部 API 完成查證與任務執行。  
## 推薦原因：
- 具有perplexity Computer 的credit（付費版），可以完成更複雜的任務。
- 有AI瀏覽器`Comet`，這個稍後我會在下一個段落介紹他。
- 學生價一個月160左右，非常便宜。
- 你買這個AI就可以用上現在市場上大多主流的AI模型（可以自己選擇），超級划算。
## 缺點：
- Pro版每個月有限制上傳檔案的數量（50個），但我目前看他好像是每日更新的...?反正就很玄啦，就很奇怪。
- 因為是新的東西還並沒有這麼的穩定，上次有爆出付費用戶的會員資格被刪除，後面雖然修好了但就是覺得沒有這麼穩定。

# Comet
::link{url="https://comet.perplexity.ai/" text="Comet" icon="https://i.namu.wiki/i/5yUU-SdijzFPCq0VHmTZLREvII1Eak0XcTU0zNLQaF8E59ehuWqciRUBdIEExTOtdaruJuxNtShJIcqSsYWbRg.svg"}
Comet 是 Perplexity 推出的「**AI 瀏覽器代理（AI browser agent）」**，不是只有內建聊天框，而是可以直接在瀏覽器裡幫你「看網頁、點按鈕、填表單、跨站完成任務」的一整套代理系統。
## Comet 是什麼
- 定位：以 Chromium 為核心的 **AI 瀏覽器**，每一個分頁背後都掛著一個「數位管家」代理，可以讀取頁面內容、理解你的自然語言指令，並在網頁上自動操作。 
- 模型：底層接上 Perplexity 的推理與代理系統，支援多步驟任務規劃、狀態追蹤與跨網站操作，像「AI 在幫你用滑鼠鍵盤上網」。
- 存取方式：需安裝 Comet AI 瀏覽器（桌面 app），啟動後就可以像用一般瀏覽器一樣瀏覽，同時在側邊欄呼叫 Comet 助理與代理模式。 
## 能做哪些事
- 自動網頁操作：幫你搜尋資料、點擊連結、切換分頁、填表單，甚至在第三方網站上完成註冊、查價、比價等任務。
- 多步驟任務代理：例如「幫我找某課程的折扣碼並套用」，Comet 會自己到多個網站搜尋、嘗試多組 coupon、直到找到可用的為止。
- 學習與開發輔助：可結合其他 AI coding 平台，充當「上位指揮官」，自動在 Replit 等平台操作、建專案、跑程式、測試結果。
- 研究與整理：自主爬多個網站、彙整資訊、做重點整理與對比，適合做產品調查、旅遊規劃、價格比對等資訊密集工作。
- 多工任務系統：可同時啟動多個代理並行工作，例如平行在不同網站或不同 AI 服務上執行任務，大幅縮短等待與切換成本。 
## 推薦原因：
- 幾乎就是Chrome再加上AI功能，不用重新學習。
- Mac版本可以直接繼承Chrome 的 cookies 和 所有紀錄。
- 手機版可以在Google 查詢時同時也詢問AI給結果。
- AI瀏覽器在處理繁瑣事情（例如填表單，抄寫東西）就不用自己用，懶人福音。
## 缺點：
- 以前曾爆出資安風險，駭客在自己網頁上進行Prompt Inject 操縱你的ai進行非法登入等等，但現在已經修復後沒有這麼容易被inject了。
- 第一次使用要設定讓他預設回Google Search Engine，有一點點麻煩（？

# Codex
::link{url="https://openai.com/codex" text="Codex" icon="https://raw.githubusercontent.com/lobehub/lobe-icons/refs/heads/master/packages/static-png/dark/codex-color.png"}
Codex 是 OpenAI 推出的「AI 程式開發代理（coding agent / coding assistant）」，不是單純一個補齊模型，而是一整套幫你規劃、撰寫、修改、測試程式碼的代理系統與工具。 
## Codex 是什麼
- 定位：雲端的 **軟體工程代理**，可以讀 repo、寫 code、開 PR、跑測試、修 bug，而不是只在一個檔案裡自動補齊。 
- 模型：底層是新版的 Codex 系列模型（如 gpt‑5 系列中的 codex 變體），專門強化長程推理、程式理解與多步驟編碼任務。 
- 存取方式：可以透過 OpenAI 的 Codex App、API SDK，或在 VS Code / GitHub 的整合中使用。
## 能做哪些事
- 功能實作：根據自然語言規格，在多檔案專案裡新增 feature、重構、改架構。 
- Bug 修復與除錯：讀錯誤訊息、log、測試輸出，找問題、提出修正 patch。 
- 測試與重構：自動產生單元測試、整理程式碼風格、抽出共用模組。
- 文件與說明：根據 codebase 產生文件、變更說明、設計規格書等。
- 跨工具工作流：可以呼叫外部「skills」，例如影像產生、Web scaffolding、文件讀寫等，完成像是「從零做一個網頁遊戲」這種大型任務。 
## 推薦原因：
- 能直接在VSCode 中直接進行編輯代碼並產生diff，隨時都可以回到上一個版本。
- 有雲端電腦可以直接連github 上的內容並操作寫代碼，也算方便...?
- 還免費送ChatGPT的服務（雖然這原本就是綑綁銷售codex & GPT plus就是了(x )
## 缺點：
- 貴，就這樣，沒有學生價，不划算。

# Gemini
::link{url="https://gemini.google.com/" text="Gemini" icon="https://raw.githubusercontent.com/lobehub/lobe-icons/refs/heads/master/packages/static-png/dark/gemini-color.png"}
Google Gemini 是 Google 推出的「**多模態 AI 助理與開發平台**」，不只是一個聊天模型，而是一套支援文字、程式碼、圖片、音訊與影片理解與生成的 AI 系統，面向一般使用者與開發者。  
## Gemini 是什麼
- 定位：Google 旗下的 **通用 AI 助理與模型家族**，從一般對話、搜尋輔助，到程式開發、雲端服務整合都有對應等級的模型與產品形態。  
- 模型：底層是 Gemini 模型系列（如 Gemini 1.x / 2.x 及其 Pro、Ultra、Flash 等變體），強調長上下文、工具使用能力與多模態輸入輸出。  
- 存取方式：可以透過 Gemini 網頁與 App 使用對話介面，也能透過 Google AI Studio、Google Cloud（Vertex AI）、或各種 IDE 外掛與 API SDK 來串接。  
## 能做哪些事
- 自然語言與搜尋輔助：回答問題、整理資訊、摘要網頁或文件，並結合 Google 搜尋結果做說明與延伸。  
- 程式開發協助：支援多種語言的程式碼生成、重構與除錯，也能理解整個檔案或專案結構並給出修改建議。  
- 多模態理解與創作：可讀取圖片、截圖或簡單影片內容，進行說明、標註、資訊萃取，並生成文字、程式碼或其他格式輸出。  
- 文件與工作流程：協助撰寫報告、簡報大綱、郵件與教學內容，搭配 Google Workspace（Docs、Sheets、Gmail 等）提升日常工作效率。  
- 開發與部署：透過 API 與雲端平台，讓開發者能將 Gemini 能力嵌入自己的網站、App、聊天機器人或內部工具中，支援權限控管與觀測。  
## 推薦原因：
- 上下文超級長，很適合用來做大專案和複雜的事情。
- 直接連接自家的各種服務，就很方便就是了。
- 一個人買會員可以共用給包含自己6個人使用，非常的划算。
- AI pro 方案有贈送 5TB 的雲端空間容量，超划算！（一個月650，也就是平均一個人110元就可以一起用了）
## 缺點：
- 我想不到什麼缺點欸，好像只有他的Code Assist 很難用而已。

# NotebookLM
::link{url="https://notebooklm.google.com/" text="NotebookLM" icon="https://static.vecteezy.com/system/resources/previews/072/969/377/non_2x/google-notebook-lm-logo-icon-high-resolution-free-vector.jpg"}
NotebookLM 是 Google 推出的「AI 筆記與研究助理」，專門幫你把自己提供的資料變成可查詢、可對話的專屬知識庫，而不是去網路亂抓資訊。  
## NotebookLM 是什麼
- 定位：雲端的 AI 筆記與研究平台，核心概念是「只根據你上傳的來源回答」，適合寫報告、做研究、整理長文件與課程資料。  
- 模型：底層以 Gemini 系列模型為主，強調多模態理解，能處理文字、PDF、簡報、YouTube 影片與音訊。  
- 存取方式：透過 NotebookLM 網頁與 App 使用，登入 Google 帳號後即可建立「筆記本」，匯入各種來源作為知識庫。  
## 能做哪些事
- 讀取與摘要：上傳 PDF、Google 文件、簡報、YouTube 或音訊逐字稿，快速生成摘要、重點條列、時間軸與 FAQ。  
- 問答與查證：你可以像聊天一樣提問，NotebookLM 會只根據你提供的來源回答，並附上引用位置或段落。  
- 知識整理與學習：自動生成學習指南、心智圖、測驗題與學習卡，幫助複習考試或梳理研究主題結構。  
- 多格式輸出：可將整理好的內容轉成 Podcast 式 audio overview、影片摘要、結構化表格，並匯出到其他工具。  
- 個人與團隊知識庫：把散落在多個檔案與連結的資料集中在同一筆記本裡，讓 NotebookLM 變成該主題的「專屬 AI 專家」。  
## 推薦原因：
- 你只需要負責上傳來源，然後他就可以依照你的來源去做很多整理和撰寫筆記提問，不常會有幻覺的產生。
- 能生成Podcast 和 Video ，根本就是創作者的福音。
## 缺點：
- 免費版好像只支持50個筆記，如果要更多要付費。

# v0.dev
::link{url="https://v0.dev/" text="v0.dev" icon="https://raw.githubusercontent.com/lobehub/lobe-icons/refs/heads/master/packages/static-png/light/v0.png"}
v0.dev（Vercel v0）是由 Vercel 推出的「AI 介面與前端程式碼生成工具」，讓你用自然語言或設計稿，就能快速產出可落地的 React / Next.js 前端介面與程式碼。  
## v0.dev 是什麼
- 定位：專注在「前端 UI 介面」的 AI 助手，更像是一個會寫 React + Tailwind + shadcn/ui 的前端工程實習生，幫你從 0 到 1 搭好初版介面。  
- 模型：以生成式 AI 結合大量現代前端範例與設計模式，可以理解自然語言描述或設計圖片，輸出結構良好、可維護的前端程式碼。  
- 存取方式：透過 v0.dev 網站登入使用，直接在瀏覽器中聊天式描述需求、預覽 UI、下載程式碼，或一鍵部署到 Vercel。  
## 能做哪些事
- UI 元件生成：輸入「幫我做一個登入頁」、「做一個含側邊欄與圖表的儀表板」之類的描述，即可自動產出對應的 React 元件與 Tailwind 樣式。  
- 完整頁面與原型：不只單一元件，也能生成整頁 layout、Landing Page、行銷頁或簡單 Web App 的雛形，用來做快速原型驗證。  
- 對話式迭代：透過聊天方式調整顏色、版面、互動行為與文案，像在跟前端討論設計修改一樣，一步步把介面修到滿意。  
- 程式碼導出與整合：可以直接複製程式碼到自己的 Next.js / React 專案，或搭配 Vercel 做一鍵部署，縮短從設計到上線的時間。  
- 進階代理能力：在較新版本中，支援類似「代理模式」，可依指令規劃多步驟任務（如建立資料庫 schema、API route 與對應 UI）來完成一個小型應用。  
## 推薦原因：
- 你直接把想法草稿畫給他或告訴他，他就會幫你把內容落實（網頁）。
- 可以直接部署，不需要自己重新學習部署，慢慢架設網站。
## 缺點：
- 一天只能回答五次（免費版），而且token 很少，要花錢才會比較方便，不被限制。

# Gamma
::link{url="https://gamma.app/" text="Gamma" icon="https://vectorseek.com/wp-content/uploads/2025/07/Gamma-Ai-Logo-PNG-SVG-Vector.png"}
Gamma 是一款「AI 簡報與內容創作平台」，主打用自然語言就能一鍵生成簡報、文件和一頁式網站，讓你專心寫內容，把排版與設計全部丟給 AI 處理。  
## Gamma 是什麼
- 定位：以「AI 代做簡報」為核心的線上創作工具，介於 PowerPoint、Notion 與簡易網站編輯器之間，適合報告、企劃書、教學簡報與提案 pitch。  
- 模型：結合生成式 AI 與自家版面引擎，能理解主題與段落結構，自動排版文字、圖片與區塊，產出風格統一、可直接上台簡報的成品。  
- 存取方式：透過 Gamma 網頁登入使用，輸入主題或貼上原始內容，就能在瀏覽器裡生成簡報，之後可匯出成 PPT、PDF 或發布為網頁連結。  
## 能做哪些事
- AI 一鍵生成簡報：輸入一段主題描述或貼上文章，Gamma 會自動拆解成多頁投影片，幫你整理標題、重點條列與示意圖片。  
- 文件與網頁排版：除了簡報，也能生成長篇文件、一頁式網站或企劃頁，直接用卡片式區塊組合出完整版面。  
- 視覺與互動元件：內建多種配色與版型、圖庫與圖示，支援圖表、嵌入影片、表單或外部服務，讓內容看起來像設計過的網站。  
- AI 編修與重寫：可以針對單一頁面或區塊下指令，請 AI 改寫文案、調整語氣、生成圖片或延伸內容，快速迭代版本。  
- 分享與匯出：完成後可用連結分享、設定觀看權限，或匯出成 PPT / PDF 再到 PowerPoint、Keynote 內做最後微調。  
## 推薦原因：
- 你如果只有簡單想法沒有內容就用gamma直接幫你製作，在水課報告製作很方便。
- 有許多種ai風格簡報可以供你使用，且還有精美動畫
## 缺點：
- 要下載成pdf或ppxt格式需要付費。

# Claude
::link{url="https://claude.ai/" text="Claude" icon="https://favicon.im/claude.ai"}
Claude 是由 Anthropic 推出的「通用 AI 助理與大型語言模型」，強調安全性與長程推理能力，擅長處理長文件、複雜思考與程式開發等深度任務。  
## Claude 是什麼
- 定位：以「安全、可信賴的 AI 夥伴」為核心設計的通用助手，可用於寫作、資料整理、策略規劃、程式開發與企業應用。  
- 模型：採用憲法式人工智慧（Constitutional AI）訓練方式，內建一套價值與安全原則，讓模型在回答時能自我調整、降低偏誤與有害內容。  
- 存取方式：可透過 Claude 官網、桌面或手機 App 使用對話介面，也能經由 Claude API 或雲端平台（如 AWS Bedrock、Vertex AI）串接到自己的服務。  
## 能做哪些事
- 深度文字工作：協助長文寫作、重寫、摘要、翻譯與風格調整，適合報告、企劃、小說與技術文件。  
- 資料整理與分析：能閱讀長篇文件、PDF 或多段輸入，整理重點、比較差異，並提出結構化分析與建議。  
- 程式開發與除錯：透過 Claude Code 等功能，支援多語言程式碼理解、重構、除錯與生成，能針對整個專案給出修改建議。  
- 推理與規劃：在策略規劃、學習路線設計、專案拆解等需要多步驟思考的場景中，提供具邏輯結構的推理與行動清單。  
- 企業與團隊應用：可整合到客服、內部知識庫、流程自動化與文件系統，讓團隊用自然語言查詢與操作既有資料。  
## 推薦原因：
- 寫代碼的好工具，好像最近也有agent 可以用的樣子。
## 缺點：
- 免費版限制太多了，所以我很少用

# DeepSeek

DeepSeek 是一家來自中國的 AI 公司與模型家族名稱，主打「高效能、低成本、偏開源」的大型語言模型，包含通用模型、推理強化模型與專門的程式碼模型。  
## DeepSeek 是什麼

- 模型：旗下包含 DeepSeek-V 系列（通用語言模型）、DeepSeek-R1（強調推理能力）、DeepSeek-Coder 系列（專攻程式碼／數學）、以及 Janus 等多模態模型。  
- 存取方式：可以透過官方網站雲端使用，也能下載開源權重自建環境，或透過各種第三方平台與 API 集成（如模型集線器、雲端推理服務）。  
## 能做哪些事
- 一般對話與寫作：處理日常問答、文章撰寫、摘要與翻譯等通用文字任務，類似 ChatGPT / Gemini 的用途。  
- 高強度推理任務：DeepSeek-R1 等模型特別針對數學題、邏輯推理與多步驟思考優化，適合解題、算法說明與系統設計討論。  
- 程式開發與數學：DeepSeek-Coder 系列專門用於程式碼理解、生成與除錯，同時在數學推理（如競程題、演算法推導）上有不錯表現。  
- 多模態應用：部分模型（如 Janus 系列）支援圖像理解與生成，可用於看圖說故事、視覺問答、簡單設計草稿。  
- 私有化與客製化：因為提供開源與自部署選項，企業或開發者可以在本地或私有雲中部署，針對自家資料做微調與權限控管。  
## 推薦原因：
- 推理能力超強，需要邏輯推理就直接用就好
## 缺點：
- 因為是中國的軟體，在政治立場上可能和普遍台灣人不同，會有違禁詞等等，在歷史上的詮釋也會不同。但這並不代表他是錯的，我們要尊重每個人的立場，在這裡我不談論更多關於立場的問題，我只想進行工具分享。

# ChatGPT
::link{url="https://chatgpt.com/" text="ChatGPT" icon="https://static.vecteezy.com/system/resources/previews/021/059/827/non_2x/chatgpt-logo-chat-gpt-icon-on-white-background-free-vector.jpg"}
ChatGPT 是 OpenAI 推出的「通用 AI 對話與創作助手」，用聊天介面就能完成問答、寫作、程式開發、資料分析、多模態理解等各種任務，是這一波生產力型 AI 的代表產品。  
## ChatGPT 是什麼
- 定位：以「萬用型 AI 助理」為核心，結合對話、搜尋、程式工具與外掛生態系，從生活問答到專業工作流程都能覆蓋。  
- 模型：底層採用 GPT 系列大型語言模型（目前為 GPT-5.x 家族），支援文字、圖片、聲音與影片等多模態輸入輸出，並提供不同能力與成本等級的子模型。  
- 存取方式：可透過 ChatGPT 網頁版、手機 App 使用，也能透過 OpenAI API、Assistants API 與各種整合插件（如 Slack、VS Code）嵌入到自己的系統。  
## 能做哪些事
- 對話與寫作：處理一般問答、文章撰寫、重寫、翻譯、摘要與風格調整，支援多語言與多種文體。  
- 程式開發與資料分析：協助理解與生成程式碼、除錯、寫測試，以及使用 Code Interpreter / Files 功能進行資料清洗、可視化與分析。  
- 多模態互動：可以看圖說明、解析截圖與文件，支援語音對話與影片內容理解，用更自然的方式溝通與提問。  
- 自訂工作流與插件：透過 Custom GPT、外掛與工具呼叫，讓 ChatGPT 幫你連動第三方服務（如搜尋、雲端硬碟、任務管理）完成複雜流程。  
- 記憶與檔案庫：能記住你的偏好與上下文，並透過檔案庫集中管理上傳的文件、試算表和圖片，跨對話使用作為長期知識來源。  
## 推薦原因：
- 就，全能型AI，什麼都能做。
## 缺點：
- 因為全能，但並不是專精，會有超多的幻覺，我一開始跟他對話，然後太多幻覺我就退訂了。

Grok 是由 Elon Musk 創立的 xAI 推出的「對話式 AI 助理與大型語言模型」，深度整合 X（原 Twitter），主打即時掌握時事、帶點毒舌幽默風格的聊天與資訊助理。

- 模型：底層是 Grok 系列大模型（目前到 Grok-3 / Grok-5 等世代），採用超大參數量與長上下文設計，並提供 DeepSearch、Think、Big Brain 等不同模式。

# ePOP
::link{url="https://apps.apple.com/us/app/epop-learn-real-life-english/id1460766549" text="ePOP" icon="https://favicon.im/epop.ai"}
ePOP 是一款主打「AI 輔助英文學習」的 App，用遊戲化關卡和智慧複習系統，幫你從單字、文法、聽力到口說循序打底，適合想扎實建立英文基礎的人。  
## ePOP 是什麼
- 定位：以「互動英語自學系統」為核心的 AI 英文學習 App，比起單純單字本，更像結合題庫、遊戲、家教老師的綜合學習平台。  
- 模型：內建 AI 英文助理與發音評分系統，會依照你的程度與錯誤紀錄調整題目難度，並針對發音細節給出即時回饋。  
- 存取方式：可在 iOS / Android 上下載使用，註冊帳號後即可進行程度測驗，由系統排出專屬學習路線與每日任務。  
## 能做哪些事
- 單字與文法練習：用選擇題、填空、例句理解等多種題型，讓你反覆練習常用單字與核心文法觀念。  
- AI 智慧複習：根據遺忘曲線與你的答題情況，自動安排何時該複習哪些字與題目，降低「背了就忘」的狀況。  
- 聽力與口說訓練：提供聽力題與跟讀練習，並用 AI 評分你的發音準確度，指出每個音標與重音的問題。  
- 實戰對話學習：透過情境式 AI 對話（例如旅行、餐廳、面試），模擬真實會話並提供即時語言回饋與替代表達。  
- 個人化學習路線：一開始的程度測驗會評估你的實力，系統會依照 CEFR 等級與弱項自動生成專屬課表與每日任務。  
## 推薦原因：
- 練習口說很好用
- 跟多鄰國比他多了更多AI輔助，會在適當的時機跟你進行解說。
## 缺點：
- 要付費，有一點貴而已，沒有付費幾乎沒辦法用這樣
