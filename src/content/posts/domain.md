# 網域介紹：

網域（Domain）就是「網路世界的地址」，用來辨識一個網站或網路服務的名稱，讓人類不用記一串數字 IP，也能找到正確的主機。 

## 什麼是「網域」

- 網域（網域名稱、域名，Domain Name）是一串用點分隔的字元，如 `example.com`。 
- 它會透過 DNS 把這個名稱「解析」成實際的 IP 位址，電腦再用這個 IP 去連線。 
- 常見比喻是「門牌號碼」：輸入對的網域，就能到對的網站門口。 

## 網域的組成

以 `www.example.com.tw` 為例： 

- `com.tw`：頂級網域（TLD），表示種類或國家，如 `.com`、`.org`、`.tw`。
- `example`：次級網域，通常是品牌或網站名稱，可自行申請，如 `pg72`。
- `www`：主機名稱／子網域，用來區分不同服務，如 `www`（網站）、`mail`（郵件）。 
- 把所有層級加起來就是完整網域名稱（FQDN）。

| 範例片段        | 名稱               | 說明                             |
|-----------------|--------------------|----------------------------------|
| `com`, `tw`     | 頂級網域           | 類型或國家/地區標示。  |
| `example`       | 次級網域           | 品牌或組織名稱。      |
| `www`, `mail`   | 子網域 / 主機名稱  | 區分不同服務。  |

## 網域、網址、IP 的關係

- 網址（URL）是完整的連結，如 `https://www.example.com/page`，包含通訊協定、網域、路徑等。 
- 網域是網址裡負責「名稱定位」的那一段，如 `example.com`。
- DNS 系統會把網域翻成 IP 位址（例如 `93.184.216.34`），讓瀏覽器知道要連到哪台伺服器。 

## 為什麼需要網域

- 好記：人類記得 `pg72.tw` 比記得一串數字 IP 容易得多。 
- 品牌：獨立網域可以建立專屬形象，如網站 `https://abcd-domain.com`、信箱 `me@abcd-domain.com` 都用同一個網域。 
- 管理：同一個網域下可以建立很多子網域（例如 `blog.pg72.tw`、`api.pg72.tw`）來拆分不同服務。

## 讓自己有更安全的電子郵件地址：
當你有了自己的網域後，可以透過轉移DNS到 `Cloudflare` 平台(這裡我不贅述要怎麼操作，詳細可以問AI。)，然後設定`電子郵件路由`，設定完後綁定自己私人電子郵件，之後你就可以自己創造電子郵件來給不同平台：

### 使用範例：
1. Discord： `discord.com@pg72.tw `
2. x(twitter)： `x@pg72.tw`
3. Facebook： `meta@pg72.tw`
4. 某個需要你留下電子郵件的平台，但你知道他有可能會發垃圾郵件： `spam@pg72.tw`
5. 公開給所有人可以發送電子郵件用的：`hi@pg72.tw`

這樣當你今天帳號被外洩時，就可以很容易的知道是哪個平台外洩的，不同平台外洩的帳密都不一樣，這樣有助於保障你的帳戶安全性。

> [!IMPORTANT]
> 如果你的網域到期沒有續費，有可能會導致你的帳號無法找回，請確保你所購買的域名會自動續費，且付款方式未過期。

## 使用自己網域電子郵件透過Gmail寄送信件：
當你挑好了電子信箱（這裡以`me@pg72.tw`作為範例）並設定上面說的`電子郵件路由`後，你可以收信了。但可以收信應該也要可以寄信啊，於是你可以跟著我一步步學習如何將這個電子信箱綁定到你的Google Gmail上寄信。
### 條件：
1. 你的帳戶不能是公司/學校帳戶，如果是，請確認機構管理員有將`應用程式密碼`功能開啟。
2. 你的Google 帳戶需要有 [雙重驗證](https://support.google.com/accounts/answer/185839?hl=zh-Hant&co=GENIE.Platform%3DDesktop)（不論是加入Passkey or 2FA驗證器。）

### 操作方式：
1. 登入這個網站 [Google應用程式密碼](https://myaccount.google.com/apppasswords) 。
2. 接下來設定一個名稱方便你辨識（這裡以 `SMTP mail`作為範例），按下產生後將彈出的16位密碼記住，接下來會使用到它。
3. 進入Gmail，點擊右上方的設定，然後選擇 `查看所有設定`。
4. 選擇 `帳戶和匯入` ，點選 `新增另一個電子郵件地址`。
5. 設定 `名稱` 和 `電子郵件`（請輸入你挑選的電子信箱）。
6. 將以下資料填寫上去：
    * 通訊卓：`587`
    * SMTP 伺服器： `smtp.gmail.com`
    * 使用者名稱： `你的 Google 電子郵件信箱`
    * 密碼：`剛剛生成的16位應用程式密碼`
    - [x] 採用 TLS 的加密連線 (建議使用)
    - [ ] 採用 SSL 的加密連線
7. 點選新增帳戶後會接下來你需要回到你的信箱，然後查看是否有一封電子郵件標題為`Gmail 確認 - 以這個地址寄送郵件 me@pg72.tw`，然後你點選裡面的驗證鏈結後，你就成功設定了！

* 如果未來寄送電子郵件想要使用這個電子郵件，請記得點選`寄件者`，並切換成你指定的電子郵件。
